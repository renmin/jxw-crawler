# jxw-crawler

一个用于自动化下载和验证图片的 Python 爬虫工具。

## 功能简介
- 支持自动遍历时间戳，查找可用图片链接
- 支持图片有效性检测（状态码、类型、大小）
- 支持图片自动下载和保存

## 使用方法
1. 修改 `crawler1.py` 中的 base URL，使用 `{ts}` 作为时间戳占位符。
2. 运行脚本自动查找并下载图片：
	```bash
	python crawler1.py
	```
3. 下载的图片会保存在 `downloads` 文件夹。

## 依赖
- requests

## 适用场景
适用于批量下载、验证图片链接有效性等自动化任务。
